Sobre o Mirror

Este mirror é atualizado automaticamente quatro vezes ao dia para o Debian, uma vez a cada hora para o Arch Linux, duas vezes ao dia para o Ubuntu e seis vezes ao dia pra Rocky Linux, garantindo que você tenha acesso às versões mais recentes dos pacotes.

Por favor, use-o de forma responsável e reporte quaisquer problemas para o e-mail de contato abaixo.

Como utilizar o Mirror

Arch Linux

Adicione a seguinte linha no começo do arquivo /etc/pacman.d/mirrorlist:

Server = https://mirrors.ic.unicamp.br/archlinux/$repo/os/$arch

Debian

Modifique a seguinte linha no arquivo /etc/apt/source.list:

deb http://ftp.br.debian.org/debian/ bookworm main non-free-firmware

para:

deb http://mirrors.ic.unicamp.br/debian/ bookworm main non-free-firmware
